# Simple Terminal RPG

ターミナルで遊べる、Python製の小さなRPGです。

## 内容

- プレイヤー名入力
- 通常敵との戦闘
- HP、攻撃力、レベル、経験値
- 回復薬
- レベルアップ
- ボス戦
- エンディング

## 遊び方

Pythonが入っている環境で、以下を実行します。

```bash
python main.py
```
